🪟 Windows 11 Optimization

Performance & Privacy Tweaks

⚠️ Important: Create a system restore point before applying these tweaks

Prevents Windows from collecting and sending your personal data to Microsoft servers. Improves privacy and slightly reduces system overhead.
🚀 Optimize Startup Programs
Disables unnecessary startup programs to reduce boot time and free up system resources. Your PC will start faster.
🎨 Optimize Visual Effects
Adjusts Windows visual effects for better performance. Disables animations and transparency effects that consume GPU resources.
⚙️ Windows Services Optimization
Disables unnecessary Windows services that run in the background. Frees up RAM and CPU resources for better performance.
🗑️ System Cleanup
Cleans temporary files, system cache, and Windows update files. Frees up disk space and improves system responsiveness.
🎯 Gaming Mode Enhancement
Optimizes Windows for gaming by prioritizing game processes and reducing background interference. Better FPS and lower input lag.

Performance & Privacy Tweaks

Optimizes Windows services for better performance. Disables non-essential services that consume system resources.
🎯 Registry Optimization
Cleans and optimizes Windows registry for better system performance. Removes invalid entries and optimizes registry structure.
🐧 Ubuntu Optimization

Linux Performance Tweaks

⚠️ Important: These tweaks require administrator privileges. Backup your system first.
🧹 Remove Snap Bloatware
Removes unnecessary snap packages and bloatware that come pre-installed with Ubuntu. Frees up disk space and improves performance.
💾 Optimize Swappiness
Reduces swappiness value to 10 for better RAM utilization. System will use physical RAM more efficiently before swapping to disk.
🚀 Enable ZRAM
Enables ZRAM for compressed RAM swap. Provides faster swap performance and better memory utilization, especially on systems with limited RAM.
⚡ Optimize Boot Time
Optimizes systemd services and boot sequence for faster startup. Reduces boot time by disabling unnecessary services during startup.
🔧 Install Essential Tweaks
Installs GNOME Tweaks, Ubuntu Restricted Extras, and other essential optimization tools for better system customization.
📦 Clean Package Cache
Cleans APT package cache and removes orphaned packages. Frees up significant disk space and keeps the system clean.
🎩 Fedora Optimization

Red Hat Performance Tweaks

⚠️ Important: These tweaks modify system repositories and configurations. Backup recommended.
📚 Enable RPM Fusion
Enables RPM Fusion free and non-free repositories. Provides access to additional software packages not available in default repos.
⚡ Optimize DNF Performance
Configures DNF package manager for faster downloads and better performance. Enables parallel downloads and optimizes cache settings.
🎵 Install Media Codecs
Installs multimedia codecs for audio and video playback. Enables support for MP3, MP4, and other proprietary formats.
🔒 Configure SELinux
Optimizes SELinux settings for better performance while maintaining security. Adjusts policies for desktop usage patterns.
🖥️ Install NVIDIA Drivers
Installs proprietary NVIDIA drivers from RPM Fusion. Provides better graphics performance and hardware acceleration support.
🧽 System Cleanup
Cleans DNF cache, removes orphaned packages, and optimizes system storage. Keeps your Fedora installation lean and fast.
